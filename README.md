## Hospital Emergency Room Analytics Dashboard | Power BI
**Overview**

End-to-end data analytics project analyzing Emergency Room (ER) operations to improve patient flow, wait times, and service quality using interactive Power BI dashboards.

**Business Problem**

Hospitals lack clear visibility into ER performance, leading to long wait times and inefficient resource allocation.
This project converts raw ER data into actionable operational insights.

**Tools**

- Power BI (DAX, Data Modeling)

- Power Query (Data Cleaning)

- CSV / Excel

**Key KPIs**

- Total ER Patients

- Average Wait Time

- Patient Satisfaction Score

- % Patients Seen Within 30 Minutes

- Admission vs Non-Admission Rate

- Department Referrals

**Dashboards**

- Monthly View: Trends by age, gender, race, time, and admissions

- Consolidated View: High-level KPIs for selected date ranges

- Patient Details: Patient-level operational analysis

**Key Insights**

- Peak hours and days drive higher wait times

- Longer waits reduce patient satisfaction

- Certain departments receive consistently high referrals

**Skills Demonstrated**

Data Cleaning • Data Modeling • DAX • KPI Design • Power BI Dashboarding • Healthcare Analytics
